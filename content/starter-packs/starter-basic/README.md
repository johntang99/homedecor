# Starter Basic Seed Pack

Use this pack for a small local laundry shop launching with:
- one primary residential pickup service
- optional drop-off
- no heavy commercial requirements

## Includes
- `booking/services.json`
- `booking/settings.json`

## How to apply
1. Copy files into `content/<siteId>/booking/`.
2. Import booking data from admin (`/api/admin/booking/import`).
3. Verify service area ZIPs and notification emails.
