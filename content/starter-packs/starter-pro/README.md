# Starter Pro Seed Pack

Use this pack for advanced clinic operations:
- consultation and follow-up scheduling
- recurring treatment plans
- corporate wellness programs

## Includes
- `booking/services.json`
- `booking/settings.json`

## How to apply
1. Copy files into `content/<siteId>/booking/`.
2. Import booking data from admin (`/api/admin/booking/import`).
3. Update ZIP coverage, blackout windows, and notification targets.
